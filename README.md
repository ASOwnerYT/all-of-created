# All of Created
A Create Mod focused modpack based on [All of Create](https://www.curseforge.com/minecraft/modpacks/aoc).

Some differences from [All of Create](https://www.curseforge.com/minecraft/modpacks/aoc):

*   Removed All of Create branding, advertisements etc
*   Added Essensials Mod, voice chat and multiplayer features
*   Added many new mobs, drugs, companions and more
